## EDA-анализ датасета Titanic
__*EDA анализ*__ (разведочный анализ) – первичный анализ данных, целями которого являются понимание структуры данных, поиск зависимостей и закономерностей, обнаружение пропусков и выбросов, а также подготовка данных к моделированию.

Анализ будет проводиться на примере **датасета Titanic** ([train.csv](https://www.kaggle.com/competitions/titanic/data?select=train.csv)), содержащего данные о каждом пассажире Титаника с пометкой, выжил ли он или нет.

Для работы с датасетом будут использоваться: 
* __Kaggle Notebook__ (для запуска кода) 
* __Python__ (для написания кода) 
* __pandas__ (библиотека для работы с датасетами, данными и .csv файлами)
* __numpy__ (библиотека для работы с инструментами линейной алгебры)

### Поиск и обработка пропусков
Для поиска и подсчета количества пропусков воспользуемся ``isnull()`` и построим новый датафрейм:

```
print("\nПропущенные значения:")
missing = train_data.isnull().sum()
missing_pct = (missing / len(train_data) * 100).round(1)
missing_df = pd.DataFrame({"Количество": missing, "Процент (%)": missing_pct})
print(missing_df[missing_df["Количество"] > 0])
```
>Итоговое количество пропусков:
> * Age - **177** (*19.9%*)
> * Cabin - **687** (*77.1%*)
> * Embark - **2** (*0.2%*)

Возможные пути решения:
1. *Удаление колонки, если >50% пропусков*
2. *Подстановка на место пропуском медианных значений, средних значений или моды*

Таким образом: 
* колонку *Age* дозаполним медианными значениями с помощью **``median()``** : ``train_data["Age"].fillna(train_data["Age"].median())``
* колонку *Cabin* не будем учитывать при анализе
* колонку *Embarked* дозаполним модой с помощью **``mode()``** : ``train_data["Embark"].fillna(train_data["Embark"].mode())``
