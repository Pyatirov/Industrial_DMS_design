# Решение задачи классификации изображений кошек и собак

Задача классификации изображений кошек и собак представляет собой бинарную задачу компьютерного зрения, где модель присваивает изображению один из двух классов: *кошка* или *собака*. Датасет содержит тысячи изображений формата .jpg с возможными посторонними объектами, что повышает сложность распознавания.

Для решения данной задачи используется особая архитектура нейронных сетей - **сверточные нейронные сети** или **CNN** (Convolutional Neural Networks).

---
## Аналитическая часть
### Полносвязная нейронная сеть

Простейшая нейронная сеть персептрон представляет собой упрощенное отображение биологической сети, состоящей из нейронов и связей между ними. Соответственно в полносвязной нейросети каждый нейрон текущего слоя связан с каждым нейроном следующего слоя. Сигнал имеет прямое распространение, то есть распространяется от входа к выходу, не имея обратной связи. Первый слой называется **входным**. Он определяет связь входов с нейронами следующего слоя. Последний выходной слой называется **выходным**., а промежуточные слои - **скрытыми**. Иногда входной слой относят к скрытым.

Эти связи между нейронами имеют **веса** $\omega_{ji}$ - связь от $j$-го нейрона к $i$-му нейрону. Когда сигнал проходит через эту связь, он умножается на ее вес.

![[img1.svg]]

Каждый нейрон можно изобразить следующим образом.

![[img2.svg]]

Здесь к нейрону подходит множество связей и на выходе формируется некоторое определенное значение, которое затем передается ко всем остальным связанным с ним нейронам. Все входы суммируются с учетом каждой связи, а затем значение этой суммы передается в функцию $f(x)$. Значение функции $f(x)$ от этой суммы называется **выходом нейрона**, а сама функция называется **функцией активации нейрона**. В концепции персептрона она выбирается как пороговая. В общем виде она записывается следующим образом: 

$$
f(x)=
\left\{
	\begin{aligned}
		a, \ \ x\geq \lambda \\
		b, \  \ x< \lambda 
	\end{aligned}
\right.
$$
Частный случай:
$$
f(x)=
\left\{
	\begin{aligned}
		1, \ \ x\geq 0 \\
		0, \  \ x<0 
	\end{aligned}
\right.
$$
Рассмотрим простейший персептрон для задачи классификации двух образов, представленных характеристиками $x_1$ и $x_2$.
![[Task3/img/img3.svg]]

Активационная функция будет выглядеть следующим образом:
$$
f(x)=
\left\{
	\begin{aligned}
		1, \ \ x\geq 0 \ \ \ \rightarrow C_1 \   \   \ \left[ x_1, x_2 \right]^T \in C_1\\ 
		-1, \  \ x<0 \ \ \ \rightarrow C_2 \   \   \ \left[ x_1, x_2 \right]^T \in C_2
	\end{aligned}
\right.
$$
Если активационная функция $f(x)$ будет давать *1* - сигнал принадлежит классу  $C_1$ , если будет давать *-1* - сигнал принадлежит классу $C_2$. Классификация векторов будет происходить следующим образом:

Запишем систему неравенств:
$$
\left\{
	\begin{aligned}
		\omega_1 x_1 + \omega_2 x_2\geq 0 \ \ \ \rightarrow C_1 \\ 
		\omega_1 x_1 + \omega_2 x_2 < 0 \ \ \ \rightarrow C_2 
	\end{aligned}
\right.
$$
Очевидно, что в значении *0* находится граница раздела классов:
$$
\omega_1 x_1 + \omega_2 x_2 = 0.
$$
Из этого выражения можно получить формулу прямой границы раздела классов:
$$x_2=-\frac{\omega_1}{\omega_2}*x_1.$$
Здесь $-\frac{\omega_1}{\omega_2}$ есть угловой коэффициент прямой.
Таким образом, мы получаем уравнение прямой, которая проходит через начало координат. Относительно этой прямой можно определить множества точек для обоих классов.
![[img4.png]]
Данная прямая называется *разделяющей прямой*. В многомерном случае она превращается в гиперплоскость и называется *разделяющей гиперплоскостью*.

$$x_2=-\frac{\omega_1}{\omega_2}*x_1-\frac{\omega_3}{\omega_2}*1,$$
$$-\frac{\omega_3}{\omega_2}=b \rightarrow \omega_3=-b*\omega_2.$$

### Сверточная нейронная сеть
#### Сканирование изображения и формирование карт признаков
Входной сигнал подается на нейрон только в пределах ограниченной зоны, например, 3х3 пикселя. Эта область скользит по изображению, и каждый новый сдвиг соответствует следующему нейрону. 

Таким образом, изображение «сканируется» одной группой нейронов с **одним набором весовых коэффициентов $\omega$**. 

![[Task3/img/img5.svg]]

Когда сканирование завершено, оно повторяется с **другим набором весов** - другой группой нейронов. 

![[Task3/img/img6.svg]]

Таких групп нейронов может быть $n$, и у каждой свой набор весов, который называется **импульсным откликом фильтра** или **ядром фильтра (*kernel*)**.

![[img7.svg]]
Формула свертки для одной карты признаков:
$$
\nu_{k,m} = \sum_{i=1}^f \sum_{j=1}^fx_{i+k,j+m} \omega_{ij} + \omega_0, \ \ k,m = 0, 1, 2, ...
$$
где $f \times f$ - размер ядра,
$k \times m$ - **координаты текущего положения ядра** при сканировании (сдвиг по строкам и столбцам).

Общее число обучаемых параметров в пределах одной группы нейронов будет равняться $f\times f + 1$, а всего при $n$ группах - 
$n(f \times f + 1)$.

Пример применения фильтров к изображению приведен ниже.

![[img8.svg]]

Здесь первый фильтр выделяет вертикальные линии, что отражено на результирующем изображении. То же самое со вторым фильтром, только с горизонтальными линиями. 

Таким образом, фильтр позволяет выделять характерные участки на изображении в соответствии с конфигурацией весовых коэффициентов.

Благодаря такому подходу нейроны каждой группы активируются тогда, когда на участке изображения появляется фрагмент, подходящий под их ядра.

![[img9.svg]]

Каждая группа имеет свое ядро. Она выделяет свои особенные характерные закономерности на изображении, проводя сканирование по всему изображению. На выходе формируется набор **карт признаков (каналов)**, количество элементов которого равно количеству групп. Каждая точка канала является выходным значением нейрона соответствующей группы.

Таким образом, значимые величины в каждой карте показывают наличие признака в строго определенном месте изображения. Точки на карте признаков будут тем больше, чем больше входной сигнал изображения будет соответствовать фильтру соответствующей группы нейронов. Используя карты признаков, следующая группа нейронов может их обрабатывать и делать дальнейшие обобщения (например, выделять более общие признаки).

---
### Карта признаков и *ReLU*

Выходная карта признаков строится как:
$$
o_{k,m} = f(\nu_{k,m},)
$$
В сверточных нейронных сетях используется функция активации ***ReLU*** (*Rectified Linear Unit*):
$$
ReLU(x)=max(0,x).
$$
* если вход x < 0, то на выходе 0;
* если вход x $\geq$ 0, то на выходе остается $x$ без изменений.

Таким образом, каждая карта признаков показывает, **где на изображении проявляется соответствующий фильтр**, а сами значения отражают «силу» этого признака.
$$
\nu_{k,m} = \sum_{i=1}^f \sum_{j=1}^fx_{i+k,j+m} \omega_{ij} + \omega_0,
$$
где C - число каналов (например, 3 для RGB).

---
### *Padding* и *stride*

В процессе свёртки влияют два важных параметра: ***padding*** и ***stride***:

* *Padding* - добавление "фиктивных" значений (обычно нулей) по границам изображения, чтобы центр ядра мог двигаться по краям.
	* При *padding* равному 0 размер карты признаков уменьшается.
	* При *padding* равному 1, шагу *stride* равному 1 и размеру ядра равному 3, размер почти сохраняется.
* *Stride* - шаг, с которым ядро сдвигается по изображению.
	* При stride = 1 сканирование идет по каждому пикселю.
	* При stride = 2 ядро "перепрыгивает" через пиксель, что уменьшает размер карты признаков и количество вычислений.

Таким образом, *padding* позволяет контролировать **размер карты признаков**, а *stride* -**грубость сканирования и количество вычислений**.

---
### Многоканальные и цветные изображения

Для многоканального изображения (например, RGB) **каждый канал свёртывается своим фильтром**, затем результаты суммируются и смещаются на bias.
$$
\nu_{k,m} = \sum_{i=1}^f \sum_{j=1}^fx_{i+k,j+m} \omega_{ij} + \omega_0,
$$
где C - число каналов (например, 3 для RGB).

После этого применяется функция активации и получается **одна выходная карта**.

![[img10.svg]]

На следующих слоях входом выступают **уже карты признаков**, а не исходное изображение, а архитектура повторяет ту же логику.

После обработки исходного изображения каждый канал фактически формирует свое новое изображение немного меньшего размера, которое равно $(N-f+1)\times(N-f+1).$

Данные правила работают для шага смещения stride = 1. При шаге равному 2, то выходное значение будет иметь размер $128:2 \times 128:2 = 64\times64$ пикселя.

---
### Пуллинг и уменьшение размерности

***Pooling*** - операция уменьшения масштаба.

Чаще всего используется **MaxPooling** с непересекающимися окнами, например 2×2:

$$
MaxPool_{i,j} = \underset{(u, v)\in W_{i,j}}{max} o_{u,v},
$$
где $W_{i,j}​$ - окно размера 2×2 на карте признаков.

![[img11.svg]]

Таким образом:
- размер карты уменьшается в 2 раза по каждому измерению,
- но при этом сохранены **наиболее сильные признаки**.

Это позволяет сети работать на разных масштабах: чем дальше в глубине, тем более **абстрактные и обобщённые** признаки извлекаются из изображения.

---
### Структура сверточной сети

Типичная архитектура:
- входное изображение (например, 32×32×3),
- несколько пар *Conv* → *ReLU* → *MaxPooling*,
- в конце - **полносвязные слои**.

![[img12.svg]]

После нескольких свёрточных уровней получается **тензор (многомерный массив) карт признаков** размера $H \times W \times C$ (высота, ширина и число карт признаков), который затем:

- растягивается в вектор (*flatten*),
- подаётся на полносвязные слои.

Чтобы подать этот тензор на полносвязные слои, его **растягивают** в вектор. Такое преобразование называется *flatten*.

---
### Полносвязные слои и софтмакс

**Конечный этап CNN в задаче классификации** обычно состоит из **двух полносвязных слоёв**:
1. Первый полносвязный слой (512 нейронов с *ReLU*) - глобально комбинирует все признаки из карт в **высокоуровневое представление**.
2. Второй полносвязный слой (2 нейрона, softmax) - преобразует это представление в вероятности принадлежности к каждому классу.
3. 
**Софтмакс** - это **математическая функция**, преобразующая вектор вещественных чисел $z$ (логитов) в **вектор вероятностей** $\sigma$ по формуле:
$$
\sigma(z_i) = \frac{e^{z_i}}{\sum_{j=0}^{K-1} e^{z_j}}, \quad i \in \{0, 1, \dots, K-1\}.
$$
где $K$ - число классов
$i$ - индекс класса
$e^{z_i}$ - экспонента от логита $z_i$,
$\sum_{j=0}^{K-1} e^{z_j}$ - полная сумма экспонент по всем классам,
$\sigma(z_i)$ - полученная вероятность, что образ принадлежит классу $i$.

Полученный вектор $\sigma$ удовлетворяет условию: $\sum_{i=0}^{K-1} \sigma(z_i)=1$.

Финальным классом выбирается тот, у которого **самая большая вероятность**.

### Подготовка изображений

Сгенерируем батчи изображений:
```
IMG_SIZE = 200                 # размер изображения
BATCH_SIZE = 32                # размер батча

datagen = ImageDataGenerator(
    rescale=1./255,            # [0, 255] -> [0, 1]
    rotation_range=20,         # случайный поворот на 20 градусов

    shear_range=0.2,           # эффект наклона

    x_col='filepath',                 # путь к файлу

    target_size=(IMG_SIZE, IMG_SIZE), # ресайз изображений
    batch_size=BATCH_SIZE,            # кол-во обрабатываемых картинок за шаг оптимизатора

)
```

Настроим слои и модель:
```
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense, Dropout

IMG_SIZE = 200
NUM_CLASSES = 2  # 0 = cat, 1 = dog
BATCH_SIZE = 32

model = Sequential([
    # 1 слой
    Conv2D(32, (3, 3), activation='relu',        # сверточный слой (числов фильтров, 
                                                 # размер окна свертки, функция активации)
           input_shape=(IMG_SIZE, IMG_SIZE, 3)),
    MaxPooling2D(2, 2),                          # пуллинг (размер окна)

    # 2 слой
    Conv2D(64, (3, 3), activation='relu'),       # сверточный слой (числов фильтров, 
                                                 # размер окна свертки, функция активации)
    MaxPooling2D(2, 2),                          # пуллинг (размер окна)

    # 3 слой
    Conv2D(128, (3, 3), activation='relu'),      # сверточный слой (числов фильтров, 
                                                 # размер окна свертки, функция активации)
    MaxPooling2D(2, 2),                          # пуллинг (размер окна)

    Flatten(),                                   # превращает тензор в вектор

    Dense(512, activation='relu'),               # полносвязный слой (число нейронов, 
                                                 # функция активации)
    Dropout(0.5),                                # регуляризация (отключение части входящих
                                                 # нейронов с вероятностью 50%)

    
    Dense(NUM_CLASSES, activation='softmax')     # полносвязный слой (число выходных классов, 
                                                 # выходные значения нейронов -> вероятности)
])

model.compile(
    optimizer='adam',                        # оптимизатор (алгоритм обновления весов)
    loss='sparse_categorical_crossentropy',  # функция потерь (считает разницу между истинной 
                                             # меткой и предсказанными вероятностями)
    metrics=['accuracy']                     # точность
)

model.summary()
```
