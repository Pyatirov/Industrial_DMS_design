# О репозитории
Данный репозиторий представляет собой **3** практические работы по дисциплине **"Проектирование систем интеллектуального анализа промышленных данных"**, выполненные студентами *СПБГТИ (ТУ)* **Пятировым Д. А.**, **Тлеумбетовым Т. Д.** и **Охотниковой Е. С.**

Каждому заданию соответствует своя папка, в которой раполагаются папка с графическим материалом (графики и рисунки), ноутбук `.ipynb` с кодом, а также отчет о проделанной работе.

Отчеты рекомендуется читать через **Obsidian**, чтобы точно отображались все графические материалы.

# Темы работ
1. *EDA*-анализ датасета Titanic
2. Решение регрессионной задачи предсказания стоимости дома с помощью модели Random Forest
3. Решение задачи классификации изображений кошек и собак
